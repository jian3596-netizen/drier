## 一、系统架构（重要：这是"双处理器"系统）

设备由**两颗芯片**通过一条串口（FPC 排线，115200）协同工作：

```
┌─────────────────────┐   UART 115200    ┌──────────────────────────┐
│  迪文 DWIN 屏          │ ◄──────────────► │  STM32F030F4P6 控制板        │
│  T5L 芯片（8051 内核）  │   FA...AF 协议    │  （Cortex-M0, 16KB Flash）   │
│  = 人机界面 + 测温/设定 │                  │  = 采集 + 控温执行            │
└─────────────────────┘                  └──────────────────────────┘
  · 显示 / 按键 / NOR 存储设定值              · 读 3 路 NTC 电阻 + 电源电压
  · NTC 电阻→温度换算（GET_Temp,带校准）       · 双区 PI 控温 + 安全联锁
  · 倒计时 / 蜂鸣 / 启停                       · 驱动 2 路加热 + 2 路风扇
```

**分工要点**：屏负责**测温（含校准系数）+ 给设定值/启停**；STM32 负责**实际的 PI 控温和安全**。
STM32 把 NTC **原始电阻值**发给屏显示；屏把**温度+设定值+状态**发回 STM32 做闭环。

**STM32 端引脚**（均为反逻辑 PWM：0=满功率/全速，99或100=关）：
- 加热1 = TIM16_CH1 / **PA6**，加热2 = TIM1_CH2 / **PA9**
- 风扇 = TIM3_CH2/CH4 = **PA7 / PB1**
- 串口 = PA2/PA3，SWD 下载口 = 原理图 **H2**

## 三、怎么编译 / 烧录

### STM32 控制板（你有 J-Link）
- 工具链：**CMake + arm-none-eabi-gcc**，纯 VSCode，无需 Keil。
- 步骤详见 **`1_STM32固件_控制板/README_编译环境.md`**（装什么、怎么 build、F5 烧录）。
- ⚠️ 第一次先编译"原样"工程确认环境通，再谈改动。

### 屏幕 8051 固件
- 需要 **Keil C51**（和 ARM 不是同一个编译器！），打开 `template.uvproj` 编译，生成 `T5L51.bin`。

### 屏幕界面
- 用 `5_工具与文档/DGUS_V7648/DGUS_V7.648.exe` 打开 `DWprj.hmi`，编译生成 `DWIN_SET`，拷到 SD 卡烧进屏。

## 五、已知坑 / 注意事项

- ⚠️ **PI 增益与安全阈值是占位值**，需台架整定（`control.c` 里标 `[TUNE]`/`[BENCH]`）。发布前务必做安全测试（拔探头/拔排线→停加热）。
- ⚠️ **缺独立硬件过温保护**：电路只有 PPTC（防过流，不防过温）。建议发热体上加机械温控开关/温度保险丝做后备（认证通常要求）。详见安全评估文档。
- ⚠️ **两份不同版本的烧屏文件**：`3_屏幕界面_DGUS/DWIN_SET` 与 `3_.../_另一版本DWIN_SET_待确认` **内容不同**（连 `T5L51.bin` 都不一样）。**需确认哪份是最终版**。
- **微动开关检测没接通**：屏端把收到的字节当作 SW1/SW2，但 STM32 没填这两字节。若产品要"插入检测"需补这条链路。
- **屏端源码注释是 GBK 编码**；STM32 端源码本次重写已转干净 ASCII。

## 六、建议学习顺序

1. 先看 **`4_硬件/Schematic_*.pdf` 原理图** → 引脚接了什么（加热×2、风扇×2、NTC×3、SWD 口）。
2. 看 **`1_STM32固件_控制板/USER_APPLICATION/control.c`** → 控温核心（双区 PI + 安全联锁，最该读的一个文件）。
3. 看 **`1_.../Core/Src/main.c`** → 很薄：初始化 + 主循环调 `Control_Update()`。
4. 看 **`2_屏幕固件_8051/USER/main.c`** → 屏端测温换算 + UI 状态机。
5. 对照 **串口协议**（下方）把两边联系起来。

### 串口协议速查
- **STM32 → 屏**（15 字节）：`FA | NTC1电阻(4) | NTC2电阻(4) | NTC3电阻(4) | AF`
- **屏 → STM32**（8 字节）：`FA | 温度1 | 温度2 | 温度3 | 运行状态 | 设定温度 | 预热标志 | AF`
